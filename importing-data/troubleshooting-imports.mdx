title: "Troubleshooting imports"
description: "What to do when rows fail to import."
---

<Note>
  This page is being written. If your import is failing, download the errors CSV from the import summary — it lists exactly which rows failed and why. Common causes: invalid email format, duplicate rows, custom field values that don't match the allowed options.

  If you're stuck, email [support@usafety.ie](mailto:support@usafety.ie) with the errors CSV attached.
</Note>
