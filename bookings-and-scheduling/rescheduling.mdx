title: "Rescheduling & cancellations"
description: "How to move or cancel an employee's booked DSE assessment."
---

Employees occasionally need to change the time they picked, or an assessor has to cancel at short notice. Usafety's in-app rescheduling experience is still being built, so the flow today depends on who needs to change what.

<Note>
  **A dedicated in-app Reschedule button in the Calendar side panel is on the roadmap.** Until it ships, use the approaches below.
</Note>

## Employee-led rescheduling

The public booking link supports a reschedule URL parameter, so if you want to let an employee pick a new slot themselves, you can send them a link in the form:

```
https://booking.usafety.ie/book/<token>?reschedule=true
```

This opens the booking form pre-loaded in reschedule mode so they can choose a new date and time within the same booking link's window. Useful for one-off changes where the employee knows which time suits them.

## Admin-led rescheduling

If you (the admin) need to move a booking — for example, the assessor can't make it and you want to pick a new slot yourself — email [info@usafety.ie](mailto:info@usafety.ie) with:

- The employee's name and email
- The current booking date and time
- The new date and time you'd like

We'll reschedule it on our side and trigger the employee and assessor notifications the same way an in-app reschedule would.

## Cancelling a booking

To cancel a booking, open it from the **Calendar** view and use the cancellation controls in the side panel. If the cancellation controls aren't visible for the booking you need to cancel, email [info@usafety.ie](mailto:info@usafety.ie) — we can cancel on your behalf and send the confirmation emails.

Cancelled bookings stay in the **Assessments** list with a *cancelled* status so you have an audit trail.

## Why this is clunky right now

Usafety's rescheduling UI is in-between versions. The underlying data model, the emails and the logic all work — but the admin-facing button in the Calendar side panel isn't exposed yet. We're prioritising it because customers are asking. If rescheduling is a daily pain point for your organisation, tell us at [info@usafety.ie](mailto:info@usafety.ie) so we can move it up the list.
