<Note>
  This page is being written. Usafety supports Microsoft 365 single sign-on and Outlook calendar sync for scheduled assessments. Both are configured from **Settings → Integrations → Microsoft 365**.

  If you need help enabling SSO for your tenant, email [support@usafety.ie](mailto:support@usafety.ie) — we can walk you through the admin consent flow.
</Note>
